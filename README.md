# Telegram Бот с Groq AI 🤖

Telegram бот на Node.js с интеграцией нейросети Groq для ответов на вопросы.

## Установка

1. Установи зависимости:
```bash
npm install
```

2. Создай файл `.env` на основе `.env.example`:
```bash
copy .env.example .env
```

3. Заполни `.env` своими ключами:
   - `BOT_TOKEN` - получи у [@BotFather](https://t.me/BotFather) в Telegram
   - `GROQ_API_KEY` - получи на [console.groq.com](https://console.groq.com/)

## Запуск

```bash
npm start
```

Для разработки с автоперезагрузкой:
```bash
npm run dev
```

## Команды бота

- `/start` - Начать общение
- `/help` - Показать справку
- `/clear` - Очистить историю диалога

Просто напиши боту любое сообщение, и он ответит с помощью AI!

## Особенности

- ✅ Сохраняет историю диалога для каждого пользователя
- ✅ Контекст последних 20 сообщений
- ✅ Индикатор печати во время обработки
- ✅ Обработка ошибок
- ✅ Кастомный system prompt с уникальным стилем общения
