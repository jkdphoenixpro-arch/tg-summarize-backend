# Режимы игры в 21 Очко

## 🃏 Блекджек (Blackjack)
**Команда:** `/blackjack`

### Описание
Классическая игра в блекджек против дилера.

### Правила
- От 2 до 6 игроков
- Все игроки играют против дилера
- Цель: набрать 21 очко или близко к этому
- Не перебрать 21!
- Дилер берет карты до 17 очков

### Условия победы
- **Победа:** Ваш счет больше счета дилера (и ≤ 21)
- **Проигрыш:** Ваш счет меньше счета дилера или вы перебрали
- **Ничья (Push):** Ваш счет равен счету дилера

### Выплаты
- **Победа:** Возврат ставки + выигрыш (x2)
- **Ничья:** Возврат ставки
- **Проигрыш:** Потеря ставки

## 🎴 21 Очко PvP (Game21)
**Команда:** `/game21`

### Описание
Игра игроков друг против друга без дилера.

### Правила
- От 2 до 6 игроков
- Игроки играют друг против друга
- Цель: набрать больше очков чем соперники
- Не перебрать 21!
- Побеждает игрок с наибольшим счетом ≤ 21

### Условия победы
- **Победа:** У вас наибольший счет среди всех игроков (≤ 21)
- **Проигрыш:** У вас не наибольший счет или вы перебрали
- **Ничья (Push):** Несколько игроков с одинаковым наибольшим счетом

### Выплаты
- **Победа:** Возврат ставки + выигрыш (x2)
- **Ничья:** Возврат ставки (при равном счете с другими)
- **Проигрыш:** Потеря ставки

---

## Общие правила

### Ставки
- Минимальная ставка: **20₽**
- Максимальная ставка: **300₽**
- Начальный баланс: **1000₽**

### Игровой процесс
1. Создайте игру командой `/blackjack` или `/game21` в группе
2. Игроки присоединяются через личку с ботом
3. Каждый игрок делает ставку (20-300₽)
4. Минимум 2 игрока для старта
5. Каждому игроку раздается по 2 карты
6. Игроки ходят по очереди (30 секунд на ход)
7. Можно взять карту или спасовать
8. После всех игроков определяются победители

### Таймер хода
- **30 секунд** на каждый ход
- При истечении времени - автоматический пас
- Цвет таймера:
  - 🟢 Зеленый: >10 секунд
  - 🟠 Оранжевый: 6-10 секунд
  - 🔴 Красный: ≤5 секунд

## Технические детали

### Изменения в коде

#### Bot (combined-bot.js)
- Команда `/game21` переименована в `/blackjack`
- Добавлена новая команда `/game21` для PvP режима
- Обе команды отправляют `gameType` на сервер

#### Server (game21/server/)
- `gameManager.js`: Добавлена поддержка `gameType` ('blackjack' или 'pvp')
- `server.js`: API `/api/create-game` принимает параметр `gameType`
- Логика определения победителей адаптирована под оба режима

#### Client (game21/client/)
- `Game.jsx`: Условный рендеринг дилера (только для blackjack)
- `Game.css`: Добавлены стили для PvP заголовка
- Автоматическое определение режима игры из `gameState.gameType`

### Структура gameState
```javascript
{
  id: "game_...",
  gameType: "blackjack" | "pvp",  // Тип игры
  players: [...],
  status: "waiting" | "playing" | "finished",
  deck: [...],
  currentPlayerIndex: 0,
  dealer: { cards: [], score: 0 } | null,  // null для PvP
  createdAt: timestamp
}
```

## FAQ

**Q: В чем разница между /blackjack и /game21?**  
A: В `/blackjack` вы играете против дилера, в `/game21` - против других игроков.

**Q: Можно ли играть одному?**  
A: Нет, минимум 2 игрока для обоих режимов.

**Q: Что если у нескольких игроков одинаковый счет в PvP?**  
A: Это считается ничьей, всем возвращаются ставки.

**Q: Дилер берет карты в PvP режиме?**  
A: Нет, в PvP режиме дилера нет вообще.

**Q: Как пополнить баланс?**  
A: Баланс автоматически сохраняется в MongoDB. Начальный баланс 1000₽.
