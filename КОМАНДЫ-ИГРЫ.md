# 🎮 Команды игры - Быстрая справка

## 📱 Telegram команды

### 🃏 Блекджек (против дилера)
```
/blackjack
```
- Классическая игра в казино
- Все игроки против дилера
- Дилер берет карты до 17
- Победа если счет > дилера

---

### 🎴 21 Очко PvP (друг против друга)
```
/game21
```
- Игроки играют друг против друга
- Дилера нет
- Побеждает игрок с максимальным счетом
- При равном счете - ничья

---

### 📊 Другие команды
```
/summarize  - Суммаризация чата
/opros      - Создать опрос
/start      - Начать работу с ботом
```

---

## 💻 Команды запуска

### Windows
```bash
# Перезапустить всё
restart-all.bat

# Запустить всё
start-all.bat

# Установить зависимости
setup.bat
```

### Вручную
```bash
# Backend
cd game21/server
npm run dev

# Frontend
cd game21/client
npm run dev

# Bot
npm run combined
```

## 🎯 Игровые действия

### В игре:
- **Взять карту** - Получить еще одну карту
- **Пас** - Остановиться и передать ход
- **Начать игру** - Запустить игру (мин. 2 игрока)

### Ставки:
- **Минимум:** 20₽
- **Максимум:** 300₽
- **Начальный баланс:** 1000₽

## 📊 API команды

### Создать игру
```bash
POST /api/create-game
Body: { "gameType": "blackjack" | "pvp" }
```

### Получить состояние
```bash
GET /api/game/:gameId
```

### Получить игрока
```bash
GET /api/player/:userId
```

### Статистика сервера
```bash
GET /api/stats
```
