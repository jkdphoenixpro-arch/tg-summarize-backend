# 🧪 Тест - Отображение счета

## Быстрый тест

### 1. Перезапустить
```bash
restart-all.bat
```

### 2. Создать игру
```
/blackjack  или  /game21
```

### 3. Присоединиться (2+ игрока)

### 4. Проверить ДО начала игры
- [ ] У других игроков НЕТ счета
- [ ] У вас НЕТ счета
- [ ] Видны только имена и ставки
- [ ] В PvP режиме виден заголовок "🎴 21 Очко - PvP"

### 5. Начать игру

### 6. Проверить ПОСЛЕ начала игры
- [ ] У других игроков ЕСТЬ счет
- [ ] У вас ЕСТЬ счет
- [ ] Счет обновляется при взятии карт
- [ ] В PvP режиме заголовок ИСЧЕЗ

---

## Тест таймера

### 1. Начать игру

### 2. Дождаться 20 секунд
- [ ] Таймер зеленый
- [ ] Нет анимации

### 3. Дождаться 10 секунд
- [ ] Таймер оранжевый
- [ ] Медленное пульсирование

### 4. Дождаться 5 секунд
- [ ] Таймер красный
- [ ] Быстрое пульсирование
- [ ] Толстая граница
- [ ] **Анимация такая же как у оранжевого**
