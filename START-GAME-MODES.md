# 🚀 Быстрый старт - Новые режимы игры

## 1️⃣ Перезапустить серверы

```bash
restart-all.bat
```

Подождите пока все запустится (~30 секунд)

---

## 2️⃣ Протестировать Blackjack

### В Telegram группе:
```
/blackjack
```

### Что должно произойти:
1. ✅ Бот отправит сообщение с кнопкой
2. ✅ Нажмите кнопку → откроется личка
3. ✅ Нажмите "Открыть игру"
4. ✅ Выберите ставку (20-300₽)
5. ✅ Дождитесь других игроков
6. ✅ Начните игру
7. ✅ **Проверьте что дилер отображается**

---

## 3️⃣ Протестировать PvP

### В Telegram группе:
```
/game21
```

### Что должно произойти:
1. ✅ Бот отправит сообщение с кнопкой
2. ✅ Нажмите кнопку → откроется личка
3. ✅ Нажмите "Открыть игру"
4. ✅ Выберите ставку (20-300₽)
5. ✅ Дождитесь других игроков
6. ✅ Начните игру
7. ✅ **Проверьте что дилера НЕТ**
8. ✅ **Проверьте заголовок "🎴 21 Очко - PvP"**

---

## 4️⃣ Проверить различия

### Blackjack
- ✅ Дилер есть
- ✅ Одна карта дилера скрыта
- ✅ Дилер берет карты после игроков
- ✅ Победа определяется относительно дилера

### PvP
- ✅ Дилера нет
- ✅ Заголовок "🎴 21 Очко - PvP"
- ✅ Побеждает игрок с максимальным счетом
- ✅ При равном счете - ничья

---

## 5️⃣ Если что-то не работает

### Проверьте логи в консоли:

**Bot (combined-bot.js):**
```
🎮 Создана игра Блекджек game_...
🎮 Создана игра 21 Очко (PvP) game_...
```

**Server (game21/server):**
```
✅ Игрок присоединился к игре
💰 Списано X₽ с баланса
⏱️ Таймер запущен для игры
```

### Если есть ошибки:
1. Перезапустите: `restart-all.bat`
2. Проверьте что все серверы запущены
3. Проверьте Redis/In-Memory хранилище
