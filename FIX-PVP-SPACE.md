# Исправление - Больше места для игроков в PvP

## 🐛 Проблема

В режиме PvP (game21) нет дилера, но блок `other-players` имел такую же высоту как в blackjack. Это создавало пустое пространство между списком игроков и карточкой текущего игрока.

---

## ✅ Решение

### Изменения в JSX

**Файл:** `game21/client/src/components/Game.jsx`

**Было:**
```jsx
<div className="other-players">
```

**Стало:**
```jsx
<div className={`other-players ${gameState.gameType === 'pvp' ? 'pvp-mode' : ''}`}>
```

**Изменение:** Добавлен условный класс `pvp-mode` для PvP режима

---

### Изменения в CSS

**Файл:** `game21/client/src/components/Game.css`

**Добавлено:**
```css
/* В PvP режиме больше места для игроков (нет дилера) */
.other-players.pvp-mode {
  max-height: 40vh;
}
```

**Медиа-запросы:**
```css
@media (max-width: 480px) {
  .other-players.pvp-mode {
    max-height: 35vh;
  }
}

@media (max-height: 700px) {
  .other-players.pvp-mode {
    max-height: 32vh;
  }
}
```

## 🔍 Детали реализации

### Условный класс
```jsx
className={`other-players ${gameState.gameType === 'pvp' ? 'pvp-mode' : ''}`}
```

- Если `gameType === 'pvp'` → добавляется класс `pvp-mode`
- Если `gameType === 'blackjack'` → только базовый класс `other-players`

### CSS каскад
```css
.other-players {
  max-height: 25vh;  /* Базовая высота */
}

.other-players.pvp-mode {
  max-height: 40vh;  /* Переопределение для PvP */
}
```
