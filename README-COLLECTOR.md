# Бот-коллектор сообщений 📝

Второй бот для сбора и сохранения сообщений из чата.

## Настройка

1. Создай нового бота у [@BotFather](https://t.me/BotFather):
   - Отправь `/newbot`
   - Придумай имя и username
   - Скопируй токен

2. Добавь токен в `.env`:
```
COLLECTOR_BOT_TOKEN=твой_токен_от_BotFather
```

3. Добавь бота в нужный чат/группу

## Запуск

```bash
npm run collector
```

Для разработки с автоперезагрузкой:
```bash
npm run collector:dev
```

## Использование

1. Добавь бота в чат где нужно собирать сообщения
2. Бот автоматически начнет сохранять все текстовые сообщения в памяти
3. Когда нужно сохранить - отправь команду `/collect`
4. Бот создаст файл с последними 100 сообщениями и отправит его в чат

## Команды

- `/start` - Информация о боте
- `/collect` - Собрать последние 100 сообщений и сохранить в файл

## Особенности

- ✅ Сохраняет до 1000 последних сообщений в памяти
- ✅ Экспортирует последние 100 сообщений по команде
- ✅ Создает текстовый файл с датой, автором и текстом
- ✅ Автоматически отправляет файл в чат
- ✅ Работает в группах и личных чатах

## Формат файла

```
[1] 29.01.2026, 20:30:15 - username: Текст сообщения

[2] 29.01.2026, 20:31:22 - username: Еще одно сообщение
```
