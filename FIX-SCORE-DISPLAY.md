# Исправление отображения счета

## 🐛 Проблемы

1. **Счет 0 в режиме ожидания** - У игроков отображался счет "0" до начала игры
2. **PvP заголовок во время игры** - Заголовок "🎴 21 Очко - PvP" оставался после начала игры
3. **Анимация красного таймера** - При критическом времени (≤5 сек) анимация отличалась от warning

---

## ✅ Исправления

### 1. Скрытие счета до начала игры

**Файл:** `game21/client/src/components/Game.jsx`

**Изменения:**
- Счет других игроков теперь показывается только если `!isWaiting`
- Счет текущего игрока также скрыт в режиме ожидания

**Код:**
```jsx
// Для других игроков
{!isWaiting && <span className="score">{player.score}</span>}

// Для текущего игрока
{!isWaiting && <span className="score large">{myPlayer.score}</span>}
```

**Результат:**
- ✅ В режиме ожидания счет не отображается
- ✅ После начала игры счет появляется
- ✅ Интерфейс выглядит чище

---

### 2. Скрытие PvP заголовка после начала игры

**Файл:** `game21/client/src/components/Game.jsx`

**Было:**
```jsx
{gameState.gameType === 'pvp' && (
  <div className="pvp-header">...</div>
)}
```

**Стало:**
```jsx
{gameState.gameType === 'pvp' && isWaiting && (
  <div className="pvp-header">...</div>
)}
```

**Результат:**
- ✅ Заголовок показывается только в режиме ожидания
- ✅ После начала игры заголовок исчезает
- ✅ Больше места для игрового поля

---

### 3. Унификация анимации таймера

**Файл:** `game21/client/src/components/Game.css`

**Было:**
```css
.turn-timer.critical {
  animation: pulse-critical 0.5s ease-in-out infinite;
  border-width: 3px;
}
```

**Стало:**
```css
.turn-timer.critical {
  animation: pulse-warning 0.5s ease-in-out infinite;
  border-width: 3px;
}
```

**Результат:**
- ✅ Красный таймер использует ту же анимацию что и оранжевый
- ✅ Анимация быстрее (0.5s вместо 1s) для большей срочности
- ✅ Граница толще (3px) для привлечения внимания

## ⏱️ Анимация таймера

### Зеленый (>10 сек)
```
Нет анимации
Спокойное состояние
```

### Оранжевый (6-10 сек)
```
animation: pulse-warning 1s
Медленное пульсирование
```

### Красный (≤5 сек)
```
animation: pulse-warning 0.5s
Быстрое пульсирование
border-width: 3px
```

**Анимация pulse-warning:**
```css
@keyframes pulse-warning {
  0%, 100% {
    transform: scale(1);
    opacity: 1;
  }
  50% {
    transform: scale(1.05);
    opacity: 0.9;
  }
}
```
