- [ ] Результаты отображаются правильно

### Баланс и выплаты

- [ ] При ничье: возврат ставки
- [ ] При проигрыше: -ставка

## 📊 Тестовые сценарии

### Сценарий 1: Blackjack - Нормальная игра
```
1. Создать игру /blackjack
2. Присоединиться 2 игрока
3. Начать игру
4. Игрок 1: взять карту, пас
5. Игрок 2: пас
6. Дилер берет карты
7. Проверить результаты
```
- [ ] Выполнено успешно

### Сценарий 2: PvP - Нормальная игра
```
1. Создать игру /game21
2. Присоединиться 3 игрока
3. Начать игру
4. Игрок 1: взять карту, пас
5. Игрок 2: пас
6. Игрок 3: взять карту, взять карту, пас
7. Проверить результаты (без дилера)
```
- [ ] Выполнено успешно

### Сценарий 3: Blackjack - Дилер перебрал
```
1. Создать игру /blackjack
2. Присоединиться 2 игрока
3. Начать игру
4. Оба игрока: пас (низкие карты)
5. Дилер берет карты и перебирает
6. Проверить что оба победили
```
- [ ] Выполнено успешно

### Сценарий 4: PvP - Ничья
```
1. Создать игру /game21
2. Присоединиться 3 игрока
3. Начать игру
4. Игрок 1: пас на 20
5. Игрок 2: пас на 20
6. Игрок 3: пас на 18
7. Проверить что 1 и 2 - ничья, 3 - проигрыш
```
- [ ] Выполнено успешно

### Сценарий 5: Таймаут
```
1. Создать игру (любую)
2. Присоединиться 2 игрока
3. Начать игру
4. Ждать 30 секунд (не делать ход)
5. Проверить автоматический пас
6. Проверить сообщение о таймауте
```
- [ ] Выполнено успешно

### Сценарий 6: Перебор
```
1. Создать игру (любую)
2. Присоединиться 2 игрока
3. Начать игру
4. Брать карты до перебора (>21)
5. Проверить автоматический проигрыш
6. Проверить что игра продолжается для других
```
- [ ] Выполнено успешно
