# Summarizetest Bot

Тестовый бот для локального тестирования команды `/summarizetest` из основного бота.

## Описание

Этот бот содержит только функционал команды `/summarizetest`, которая:
- Работает в личных сообщениях с любым пользователем
- Собирает историю сообщений из целевого чата через userbot
- Анализирует сообщения с помощью Groq AI
- Возвращает развернутое саммари чата

## Установка

1. Установи зависимости:
```bash
npm install
```

2. Скопируй `.env.example` в `.env`:
```bash
cp .env.example .env
```

3. Заполни `.env` файл:
   - `BOT_TOKEN` - токен бота от @BotFather
   - `GROQ_API_KEY` - API ключ от Groq
   - `TELEGRAM_API_ID` - API ID от https://my.telegram.org
   - `TELEGRAM_API_HASH` - API Hash от https://my.telegram.org
   - `TELEGRAM_SESSION` - строка сессии userbot
   - `TARGET_CHAT_ID` - ID чата для анализа

## Запуск

```bash
npm start
```

Или с автоперезагрузкой:
```bash
npm run dev
```

## Команды бота

- `/start` - Приветствие и инструкции
- `/status` - Проверка статуса бота
- `/summarizetest` - Анализ сообщений из целевого чата

## Как использовать

1. Запусти бота
2. Напиши боту в личку `/start`
3. Используй `/summarizetest` для анализа чата
4. Бот соберет последние сообщения и вернет саммари

## Отличия от основного бота

- Работает с любым пользователем (не только с владельцем)
- Содержит только команду `/summarizetest`
- Минимальный функционал для тестирования
- Не влияет на основной бот

## Требования

- Node.js 18+
- Telegram Bot Token
- Groq API Key
- Настроенный userbot (TELEGRAM_SESSION)
- Доступ к целевому чату
