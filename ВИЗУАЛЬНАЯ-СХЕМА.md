# 🎨 Визуальная схема режимов

## 🎮 Две команды - два режима

```
┌─────────────────────────────────────────────────────┐
│                  TELEGRAM BOT                       │
└─────────────────────────────────────────────────────┘
                        │
        ┌───────────────┴───────────────┐
        │                               │
        ▼                               ▼
┌───────────────┐               ┌───────────────┐
│  /blackjack   │               │   /game21     │
│               │               │               │
│ 🃏 С дилером  │               │ 🎴 PvP режим  │
└───────────────┘               └───────────────┘
```

---

## 🃏 Blackjack - Против дилера

```
        ┌─────────┐
        │  ДИЛЕР  │ ← Берет карты до 17
        └────┬────┘
             │
    ┌────────┼────────┐
    │        │        │
    ▼        ▼        ▼
┌────────┐┌────────┐┌────────┐
│Игрок 1 ││Игрок 2 ││Игрок 3 │
└────────┘└────────┘└────────┘

Все играют против дилера!
```

### Пример игры:
```
Игрок 1: 18 очков → 18 < 20 → ❌ Проигрыш
Игрок 2: 21 очко  → 21 > 20 → 🏆 Победа
Игрок 3: 20 очков → 20 = 20 → 🤝 Ничья
Дилер:   20 очков
```

---

## 🎴 Game21 PvP - Друг против друга

```
┌────────┐  ┌────────┐  ┌────────┐
│Игрок 1 │  │Игрок 2 │  │Игрок 3 │
└───┬────┘  └───┬────┘  └───┬────┘
    │           │           │
    └───────────┼───────────┘
                │
                ▼
        Сравнение друг с другом
        Побеждает максимум!
```

### Пример игры:
```
Игрок 1: 18 очков → 18 < 21 → ❌ Проигрыш
Игрок 2: 21 очко  → 21 = max → 🏆 Победа
Игрок 3: 19 очков → 19 < 21 → ❌ Проигрыш
```

---

## 📊 Сравнение

```
┌──────────────┬──────────────┬──────────────┐
│   Параметр   │  Blackjack   │   Game21     │
├──────────────┼──────────────┼──────────────┤
│   Дилер      │      ✅      │      ❌      │
│   Соперник   │    Дилер     │   Игроки     │
│   Победа     │  > дилера    │  Максимум    │
│   Ничья      │  = дилеру    │  = максимум  │
└──────────────┴──────────────┴──────────────┘
```

## 🎯 Как играть

```
1. Команда в группе
   ↓
   /blackjack  или  /game21
   ↓
2. Кнопка в личке
   ↓
   "Открыть игру"
   ↓
3. Выбор ставки
   ↓
   20-300₽
   ↓
4. Ожидание игроков
   ↓
   Минимум 2
   ↓
5. Начало игры
   ↓
   Раздача карт
   ↓
6. Ходы игроков
   ↓
   Взять карту / Пас
   ↓
7. Результат
   ↓
   Победа / Ничья / Проигрыш
```

## 💰 Выплаты

```
┌─────────────┬──────────┬──────────┐

├─────────────┼──────────┼──────────┤
│  Победа 🏆  │  100₽    │  +100₽   │
│  Ничья 🤝   │  100₽    │   ±0₽    │
│  Проигрыш❌ │  100₽    │  -100₽   │
└─────────────┴──────────┴──────────┘
```

## ⏱️ Таймер хода

```
30 секунд на ход

🟢 >10 сек  → Зеленый
🟠 6-10 сек → Оранжевый
🔴 ≤5 сек   → Красный

Время вышло → Автоматический пас
```

## 🔄 Процесс игры

### Blackjack
```
Ожидание → Раздача → Ходы игроков → Ход дилера → Результат
  (2+)     (2 карты)    (30с/ход)      (до 17)    (выплаты)
```

### PvP
```
Ожидание → Раздача → Ходы игроков → Результат
  (2+)     (2 карты)    (30с/ход)    (выплаты)
                                         ↑
                                  Без хода дилера!
```
