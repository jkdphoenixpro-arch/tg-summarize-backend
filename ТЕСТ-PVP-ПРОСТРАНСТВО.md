### 1. Создать игру
```
/blackjack
```

### 2. Присоединиться 4+ игрока

### 3. Начать игру

### 4. Проверить
- [ ] Дилер виден вверху

## Тест PvP (должно быть больше места)

### 1. Создать игру
```
/game21
```

### 2. Присоединиться 4+ игрока

### 3. Начать игру

### 4. Проверить
- [ ] Дилера НЕТ

- [ ] Нет пустого пространства между игроками и вашей карточкой
