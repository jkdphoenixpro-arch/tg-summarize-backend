# Объединенный бот: AI + Коллектор 🤖📝

Один бот который собирает сообщения из группы И общается с тобой в личке.

## Возможности

✅ Собирает сообщения из группы автоматически
✅ Общается с тобой в личке (токсичный AI)
✅ Анализирует переписку из группы по команде
✅ Работает только с владельцем

## Настройка

1. **Добавь в `.env`:**
```env
BOT_TOKEN=твой_основной_токен
GROQ_API_KEY=твой_groq_ключ
TARGET_CHAT_ID=-1002438653104
```

2. **Узнай ID чата:**
   - Добавь бота в группу
   - Напиши что-нибудь в группе
   - Посмотри в логах: `Начал собирать из чата -1002438653104`
   - Скопируй этот ID в `.env` как `TARGET_CHAT_ID`

## Запуск

```bash
npm run combined
```

Для разработки:
```bash
npm run combined:dev
```

## Использование

### В группе:
- Добавь бота в группу
- Он автоматически собирает все сообщения
- Молча работает в фоне

### В личке с ботом:

**Команды:**
- `/start` - Информация о боте
- `/analyze` - Проанализировать последние 50 сообщений из группы
- `/status` - Сколько сообщений собрано
- `/clear` - Очистить память бота

**Общение:**
- Просто напиши боту что-нибудь
- Он ответит в токсичном стиле
- Использует AI для анализа твоих слабых мест

## Примеры

**Анализ группы:**
```
Ты: /analyze
Бот: брррааат... щас гляну что там эти долбоёбы пишут 💀
     [токсичный анализ переписки]
```

**Обычное общение:**
```
Ты: Привет
Бот: чё за хуйня... опять ты... 🤬 [токсичный ответ]
```

## Особенности

- 📊 Хранит до 500 последних сообщений из группы
- 🔒 Работает только с владельцем (кто первый написал /start)
- 🤖 Использует Groq AI для ответов
- 💾 Все в памяти (при перезапуске сбрасывается)

## Отличия от раздельных ботов

**Было:**
- `index.js` - только AI общение
- `collector-bot.js` - только сбор сообщений

**Стало:**
- `combined-bot.js` - всё в одном!
- Один токен, один бот
- Проще в использовании
