# ✅ Завершено: UI для баланса и выбора ставки

## 🎯 Что реализовано

Полностью завершена система баланса и ставок для игры 21 Очко:

### 1️⃣ Лобби с выбором ставки
- 💰 Карточка с балансом игрока (загружается с сервера)
- 🎲 Выбор ставки от 20₽ до 300₽
- ➕➖ Кнопки быстрого изменения (+10/-10)
- ⌨️ Ручной ввод суммы
- ✅ Проверка достаточности средств
- 🚫 Блокировка при недостатке баланса

### 2️⃣ Отображение в игре
- 💵 Баланс вверху экрана
- 🏷️ Ставка каждого игрока
- 🎉 Результат с суммой выигрыша/проигрыша
- 🔄 Автоматическое обновление баланса

### 3️⃣ Backend
- 🔌 API endpoint `/api/player/:userId`
- ✔️ Проверка баланса и лимитов
- 💾 Сохранение в MongoDB (опционально)
- 🔄 Обновление баланса после игры

## 🔧 Технические детали

### API
```javascript
// Получить баланс игрока
GET /api/player/:userId

// Response
{
  "success": true,
  "player": {
    "userId": "123456",
    "username": "Player",
    "balance": 1000,
    "gamesPlayed": 5,
    "gamesWon": 3
  }
}
```

### Socket.io
```javascript
// Присоединиться к игре со ставкой
socket.emit('join_game', {
  gameId: 'game123',
  userId: '123456',
  username: 'Player',
  bet: 100  // Новое поле
});
```

### Структура игрока
```javascript
{
  userId: '123456',
  username: 'Player',
  cards: [...],
  score: 18,
  status: 'active',
  bet: 100,        // Ставка
  balance: 1000,   // Баланс
  result: 'win'    // Результат
}
```
