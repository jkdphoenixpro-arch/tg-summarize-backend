# ⚡ ЗАПУСК - Быстрая инструкция

## 1. Перезапустить серверы

```bash
restart-all.bat
```

Подождите 30 секунд пока всё запустится.

---

## 2. Проверить что работает

### В консоли должно быть:

**Bot:**
```
✅ Бот запущен
```

**Server:**
```
🎮 Game server запущен на порту 3001
```

**Client:**
```
➜ Local: http://localhost:5173/
```

---

## 3. Протестировать команды

### В Telegram группе:

```
/blackjack
```
→ Должна создаться игра с дилером

```
/game21
```
→ Должна создаться игра PvP (без дилера)

---

## 4. Проверить игру

### Blackjack:
- ✅ Дилер отображается
- ✅ Одна карта скрыта
- ✅ Дилер берет карты

### PvP:
- ✅ Дилера нет
- ✅ Заголовок "🎴 21 Очко - PvP"
- ✅ Игроки играют друг против друга

---

## 5. Если не работает

### Проверьте:
1. Все серверы запущены
2. Нет ошибок в консоли
3. Redis/In-Memory работает

### Перезапустите:
```bash
restart-all.bat
```

---

## ✅ Готово!

Если обе команды работают - всё отлично! 🎉
