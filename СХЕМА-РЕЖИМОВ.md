# 🎮 Визуальная схема режимов игры

## Архитектура системы

```
┌─────────────────────────────────────────────────────────┐
│                    TELEGRAM BOT                         │
│                  (combined-bot.js)                      │
└─────────────────────────────────────────────────────────┘
                           │
                           │
        ┌──────────────────┴──────────────────┐
        │                                     │
        ▼                                     ▼
┌───────────────┐                    ┌───────────────┐
│  /blackjack   │                    │   /game21     │
│               │                    │               │
│ Против дилера │                    │  PvP режим    │
└───────┬───────┘                    └───────┬───────┘
        │                                     │
        │  gameType: "blackjack"             │  gameType: "pvp"
        │                                     │
        └──────────────────┬──────────────────┘
                           │
                           ▼
        ┌──────────────────────────────────────┐
        │      GAME SERVER (server.js)         │
        │   POST /api/create-game              │
        │   { gameType: "blackjack" | "pvp" }  │
        └──────────────────┬───────────────────┘
                           │
                           ▼
        ┌──────────────────────────────────────┐
        │    GAME MANAGER (gameManager.js)     │
        │                                      │
        │  createGame(gameType)                │
        │  startGame()                         │
        │  determineWinners()                  │
        └──────────────────┬───────────────────┘
                           │
                           ▼
        ┌──────────────────────────────────────┐
        │         GAME STATE                   │
        │                                      │
        │  {                                   │
        │    gameType: "blackjack" | "pvp"    │
        │    dealer: {...} | null              │
        │    players: [...]                    │
        │  }                                   │
        └──────────────────┬───────────────────┘
                           │
                           ▼
        ┌──────────────────────────────────────┐
        │      CLIENT (Game.jsx)               │
        │                                      │
        │  if (gameType === "blackjack")       │
        │    → Показать дилера                 │
        │  else                                │
        │    → Показать PvP заголовок          │
        └──────────────────────────────────────┘
```

## Поток создания игры

### Blackjack (против дилера)

```
1. Пользователь → /blackjack в группе
                    ↓
2. Bot → POST /api/create-game { gameType: "blackjack" }
                    ↓
3. GameManager → createGame("blackjack")
                    ↓
4. GameState → {
                  gameType: "blackjack",
                  dealer: { cards: [], score: 0 },
                  players: []
                }
                    ↓
5. Client → Показывает дилера
```

### PvP (игроки друг против друга)

```
1. Пользователь → /game21 в группе
                    ↓
2. Bot → POST /api/create-game { gameType: "pvp" }
                    ↓
3. GameManager → createGame("pvp")
                    ↓
4. GameState → {
                  gameType: "pvp",
                  dealer: null,
                  players: []
                }
                    ↓
5. Client → Показывает PvP заголовок
```

## Условия победы

### Blackjack
```
IF перебор (>21)
  → Проигрыш ❌

ELSE IF дилер перебрал
  → Победа 🏆

ELSE IF счет > счета дилера
  → Победа 🏆

ELSE IF счет = счету дилера
  → Ничья 🤝

ELSE
  → Проигрыш ❌
```

### PvP
```
IF перебор (>21)
  → Проигрыш ❌

ELSE IF счет = максимальному
  IF только один с таким счетом
    → Победа 🏆
  ELSE
    → Ничья 🤝

ELSE
  → Проигрыш ❌
```

## Ключевые различия в коде

### createGame()
```javascript
// Blackjack
dealer: { cards: [], score: 0 }

// PvP
dealer: null
```

### startGame()
```javascript
// Blackjack
game.dealer.cards = [deck.pop(), deck.pop()];

// PvP
// Дилеру карты не раздаются
```

### moveToNextPlayer()
```javascript
// Blackjack
await this.dealerTurn(game);

// PvP
// Хода дилера нет
```

### determineWinners()
```javascript
// Blackjack
if (player.score > dealerScore) → win

// PvP
if (player.score === maxScore) → win
```
