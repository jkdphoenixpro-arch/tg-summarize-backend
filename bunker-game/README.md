# 🛡️ AI-Random Bunker - Инструкция по использованию

## 📖 Описание игры

AI-Random Bunker — это улучшенная версия классической игры "Бункер", где игроки (4-6 человек) борются за выживание в бункере после апокалипсиса. Игра полностью рандомизирована: ИИ генерирует уникальные сценарии, роли и события на лету.

### 🎲 Особенности:
- **Рандомный уровень непредсказуемости** (1-5) для каждой игры
- **AI-генерация** сценариев, ролей и аргументов через модель `openai/gpt-oss-120b`
- **Скрытые механики** и неожиданные твисты
- **Время партии**: 25-50 минут
- **Победа**: выжить вдвоём + максимум очков

## 🚀 Как начать игру

### 1. Создание игры (в группе)
```
/randombunker
```

Бот создаст сообщение с информацией об игре:
- 🎲 Уровень непредсказуемости (генерируется случайно от 1 до 5)
- 👥 Количество игроков (4-6)
- ✅ Кнопка "Присоединиться"
- ❌ Кнопка "Выйти"

### 2. Сбор игроков

Игроки нажимают кнопку **"Присоединиться"** чтобы войти в игру.

**Автостарт происходит когда:**
- Набралось 6 игроков (максимум)
- Прошло 5 минут и набралось минимум 4 игрока

**Отмена игры:**
- Если через 5 минут меньше 4 игроков - игра отменяется

### 3. Начало игры

После старта:
1. ИИ генерирует уникальный сценарий апокалипсиса
2. Каждому игроку назначается случайная роль
3. Бот отправляет уведомления в личку каждому игроку

## 🎮 Игровой процесс

### Узнать свою роль (в личке бота)
```
/myrole
```

Бот отправит тебе:
- 🎭 Название роли
- 📝 Описание навыков и особенностей
- 🔒 Секретную информацию

**ВАЖНО:** Роль видна только тебе!

### Представить свою роль (в группе)
```
/role
```

ИИ сгенерирует убедительный аргумент для защиты твоей позиции в бункере.
- 30% шанс что аргумент будет содержать блеф
- Используй это чтобы убедить других игроков

### Голосование за исключение (в группе)
```
/vote
```

Бот создаст кнопки для голосования за каждого игрока.
Каждый может проголосовать один раз.

### Завершить голосование (в группе)
```
/endvote
```

Бот подсчитает голоса и исключит игрока с максимальным количеством голосов.

**После исключения:**
- Раскрывается роль исключённого
- Показывается количество голосов
- Если осталось 2 игрока - игра завершается

### Остановить игру (только админ группы)
```
/stopbunker
```

Принудительно завершает текущую игру.

## 📊 Уровни непредсказуемости

Каждая игра генерируется с случайным уровнем непредсказуемости:

| Уровень | Название | Описание |
|---------|----------|----------|
| 1 | Стандартный | Классический апокалипсис с предсказуемыми событиями |
| 2 | Умеренный | Добавлены неожиданные повороты и мутации |
| 3 | Хаотичный | Множество случайных событий и скрытых механик |
| 4 | Безумный | Полный хаос с постоянными твистами и сюрпризами |
| 5 | Абсурдный | Максимальная непредсказуемость, реальность ломается |

Чем выше уровень - тем больше:
- Скрытых аномалий в сценарии
- Шанс мутации ролей
- Неожиданных событий

## 🎯 Стратегия игры

### Для выживания:
1. **Изучи свою роль** - используй секретную информацию
2. **Блефуй разумно** - не переборщи (ИИ может намекнуть группе)
3. **Анализируй других** - кто полезен, а кто саботажник?
4. **Формируй альянсы** - но помни о предательстве
5. **Следи за ресурсами** - они влияют на исход игры

### Скрытые механики:
- Некоторые роли - саботажники (победа: <3 игроков)
- Роли могут мутировать в процессе игры
- Ресурсы тратятся неочевидно
- Блеф >50% - ИИ намекнёт группе
- Исключение определённых ролей триггерит события

## 🔧 Технические требования

### Для работы игры нужно:
1. Бот должен быть добавлен в группу
2. В `.env` файле должен быть указан `GROQ_API_KEY`
3. Игроки должны начать диалог с ботом (для получения роли в личку)

### Модель ИИ:
Используется модель `openai/gpt-oss-120b` через Groq API для генерации:
- Сценариев апокалипсиса
- Описаний ролей
- Аргументов для защиты

## ❓ Частые вопросы

**Q: Почему я не получил роль в личке?**
A: Начни диалог с ботом командой `/start` в личных сообщениях.

**Q: Можно ли играть с меньше чем 4 игроками?**
A: Нет, минимум 4 игрока для баланса игры.

**Q: Как узнать кто саботажник?**
A: Это скрытая механика. Анализируй поведение и аргументы игроков.

**Q: Можно ли запустить несколько игр одновременно?**
A: Да, в разных группах могут идти разные игры.

**Q: Что делать если игра зависла?**
A: Админ группы может использовать `/stopbunker` для остановки.
