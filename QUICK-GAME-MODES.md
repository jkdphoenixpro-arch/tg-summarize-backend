# 🎮 Быстрая шпаргалка - Режимы игры

## Две команды - два режима!

### 🎴 `/game21` - PvP
Игроки друг против друга, без дилера

### 🃏 `/blackjack` - Классика
Все против дилера

---

## Как играть

```
1. В группе: /game21 или /blackjack
2. Нажми кнопку в личке
3. Сделай ставку (20-300₽)
4. Жди других игроков (мин. 2)
5. Начни игру!
```

---

## Главные отличия

| | `/game21` | `/blackjack` |
|---|---|---|
| **Дилер** | ❌ | ✅ |
| **Против кого** | Игроки | Дилер |
| **Победа** | Больше всех | Больше дилера |

---

## Примеры

### PvP (`/game21`)
```
Игрок 1: 18 → ❌
Игрок 2: 21 → 🏆 Победа!
Игрок 3: 19 → ❌
```

### Blackjack (`/blackjack`)
```
Игрок 1: 18 → ❌
Игрок 2: 21 → 🏆 Победа!
Дилер: 20
```

---

## Выплаты

- 🏆 **Победа:** +100% (ставка × 2)
- 🤝 **Ничья:** Возврат ставки
- ❌ **Проигрыш:** -100%

---

## Правила

- ⏱️ **30 секунд** на ход
- 💰 **20-300₽** ставка
- 🎯 **Не больше 21!**
- 👥 **2-6 игроков**

---

## Тестирование

```bash
# Перезапустить
restart-all.bat

# В Telegram группе

```
