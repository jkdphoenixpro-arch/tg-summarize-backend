# Исправление - Скрытие PvP заголовка

## 🐛 Проблема

PvP заголовок "🎴 21 Очко - PvP" оставался на экране после начала игры, занимая лишнее место.

---

## ✅ Решение

**Файл:** `game21/client/src/components/Game.jsx`

**Было:**
```jsx
{gameState.gameType === 'pvp' && (
  <div className="pvp-header">
    <h2>🎴 21 Очко - PvP</h2>
    <p className="pvp-subtitle">Играйте друг против друга!</p>
  </div>
)}
```

**Стало:**
```jsx
{gameState.gameType === 'pvp' && isWaiting && (
  <div className="pvp-header">
    <h2>🎴 21 Очко - PvP</h2>
    <p className="pvp-subtitle">Играйте друг против друга!</p>
  </div>
)}
```

**Изменение:** Добавлено условие `&& isWaiting`
