# ❓ Часто задаваемые вопросы (FAQ)

## Общие вопросы

### Что такое игра 21 Очко?
Это многопользовательская карточная игра (Blackjack), интегрированная в Telegram через WebApp. От 2 до 6 игроков могут играть одновременно в реальном времени.

### Нужно ли устанавливать что-то на телефон?
Нет! Игра работает прямо в Telegram через WebApp. Просто нажми кнопку и играй.

### Можно ли играть одному?
Нет, нужно минимум 2 игрока. Но ты можешь открыть игру в двух браузерах для теста.

## Установка и настройка

### Нужен ли Redis?
Да, Redis обязателен для хранения состояния игр. Без него игра не будет работать.

### Как установить Redis на Windows?
Самый простой способ - через Docker:
```bash
docker run -d -p 6379:6379 redis
```

Или через WSL:
```bash
wsl
sudo apt install redis-server
redis-server
```

### Что делать если Redis не подключается?
```bash
# Проверь что Redis запущен
redis-cli ping

# Должно вернуть: PONG
# Если нет - перезапусти Redis
```

### Нужен ли HTTPS для локального тестирования?
Для тестирования в браузере - нет. Для Telegram WebApp - да. Используй ngrok для локального HTTPS.

## Использование

### Как создать игру?
В группе Telegram напиши `/game21` и нажми кнопку "🎮 Играть".

### Как присоединиться к игре?
Нажми на кнопку "🎮 Играть" в сообщении бота. Все кто нажмут на эту кнопку присоединятся к одной игре.

### Сколько времени есть на ход?
Сейчас нет ограничения по времени. Игроки ходят по очереди.

### Что если игрок вышел из игры?
Игра продолжится без него. Его ход будет пропущен.

### Можно ли играть в нескольких играх одновременно?
Да, каждая игра имеет уникальный ID. Можешь создать несколько игр в разных группах.

## Правила игры

### Как считаются очки?
- Карты 2-10: номинал карты
- J, Q, K: 10 очков
- A: 11 или 1 (автоматически выбирается лучший вариант)

### Что такое "перебор"?
Если сумма твоих карт больше 21 - ты проиграл (перебор/bust).

### Когда дилер берет карты?
После того как все игроки завершили свои ходы (взяли карты или спасовали).

### Кто выигрывает?
- У кого больше очков чем у дилера (но не больше 21)
- Если дилер перебрал, а ты нет
- Ничья если у тебя и дилера одинаковое количество очков

### Что такое "Blackjack"?
21 очко с первых двух карт (туз + карта на 10 очков). В текущей версии это просто 21 очко.

## Технические вопросы

### Почему игра не загружается?
1. Проверь что backend запущен (порт 3001)
2. Проверь что frontend запущен (порт 5173)
3. Проверь что Redis работает
4. Проверь переменные окружения

### Ошибка "Cannot connect to Redis"
```bash
# Убедись что Redis запущен
redis-cli ping

# Проверь REDIS_URL в server/.env
REDIS_URL=redis://localhost:6379
```

### Ошибка "Socket.io connection failed"
1. Проверь `VITE_SOCKET_URL` в `client/.env`
2. Убедись что backend запущен
3. Проверь CORS настройки в `server/server.js`

### Telegram WebApp показывает пустой экран
1. Проверь консоль браузера (F12)
2. Убедись что используешь HTTPS
3. Проверь что `WEBAPP_URL` правильный в корневом `.env`

### Как посмотреть логи?
Логи выводятся в консоль где запущен сервер. Используй `console.log` для отладки.

## Разработка

### Как добавить новую функцию?
1. Backend: `game21/server/gameManager.js` - логика
2. Frontend: `game21/client/src/components/` - UI
3. Socket events: добавь в `server.js` и `App.jsx`

### Как изменить дизайн?
Все стили в `game21/client/src/components/*.css`. Используй CSS переменные для цветов.

### Как добавить звуки?
```javascript
const sound = new Audio('/sounds/card.mp3');
sound.play();
```

### Как добавить анимации?
Используй CSS transitions и animations в компонентах.

### Можно ли использовать другую базу данных вместо Redis?
Да, но нужно переписать `gameManager.js`. Redis выбран за скорость и простоту.

## Деплой

### Где хостить игру?
- **Backend:** Railway, Render, Heroku
- **Frontend:** Vercel, Netlify
- **Redis:** Upstash, Redis Cloud

### Сколько стоит хостинг?
Можно использовать бесплатные планы:
- Vercel: бесплатно
- Railway: $5/месяц (с бесплатным trial)
- Upstash: бесплатный план есть

### Нужен ли домен?
Нет, хостинги предоставляют свои поддомены. Но свой домен можно подключить.

### Как обновить игру после деплоя?
```bash
# Frontend (Vercel)
cd client
vercel --prod

# Backend (Railway)
cd server
railway up
```

## Производительность

### Сколько игр может работать одновременно?
Зависит от сервера. На бесплатном плане Railway - около 10-20 активных игр.

### Как долго хранится состояние игры?
По умолчанию 1 час (3600 секунд). Можно изменить в `gameManager.js`.

### Можно ли увеличить количество игроков?
Да, измени проверку в `gameManager.js`:
```javascript
if (game.players.length >= 10) { // было 6
```

### Как очистить старые игры?
Redis автоматически удаляет игры через TTL (1 час). Или вручную:
```bash
redis-cli FLUSHALL
```

## Безопасность

### Безопасна ли игра?
Да, но для продакшна рекомендуется:
1. Добавить аутентификацию
2. Валидацию данных
3. Rate limiting
4. HTTPS обязательно

### Можно ли читерить?
В текущей версии - теоретически да, так как клиент знает все карты. Для продакшна нужно скрывать карты дилера на сервере.

### Как защититься от DDoS?
Используй rate limiting и Cloudflare перед сервером.
