## Цветовая схема

### Лобби
- **Фон**: Темный (Telegram theme)
- **Карточка баланса**: Градиент фиолетовый (#667eea → #764ba2)
- **Кнопки**: Синий (Telegram button color)
- **Текст**: Белый/Серый

### Игра
- **Фон**: Темный
- **Карточка игрока**: Градиент фиолетовый
- **Дилер/Другие**: Темно-серый
- **Победа**: Зеленый (#4caf50)
- **Проигрыш**: Красный (#f44336)
- **Ничья**: Серый (#9e9e9e)

## Адаптивность

### Мобильные устройства (< 480px)
- Уменьшенные отступы
- Карточки игроков в 2 колонки
- Крупные кнопки для удобства

### Планшеты (480px - 768px)
- Карточки игроков в 3 колонки
- Стандартные размеры

### Десктоп (> 768px)
- Максимальная ширина 400px по центру
- Все элементы видны без прокрутки

## Анимации

### Текущие
- Плавное появление элементов
- Hover эффекты на кнопках
- Transition при изменении состояния

### Планируемые
- Анимация раздачи карт
- Анимация обновления баланса
- Конфетти при победе
- Вибрация при проигрыше

## Иконки

- 💰 Баланс
- 🎴 Игра
- 🎩 Дилер
- 🃏 Взять карту
- ✋ Пас
- 🎉 Победа
- 😔 Проигрыш
- 🤝 Ничья
- 💥 Перебор
- 🏆 Победитель

## Типографика

- **Заголовки**: 2.5rem, жирный
- **Баланс**: 2.5rem, жирный
- **Счет**: 1.5-2rem, жирный
- **Текст**: 1rem, обычный
- **Подсказки**: 0.85rem, светлый

## Состояния кнопок

### Активная
- Полная непрозрачность
- Hover эффект
- Курсор pointer

### Неактивная
- Opacity: 0.5
- Нет hover эффекта
- Курсор not-allowed

### Загрузка
- Spinner анимация
- Текст "Загрузка..."
- Кнопка заблокирована

## Сообщения об ошибках

```
┌─────────────────────────────────┐
│  ❌ Недостаточно средств        │
└─────────────────────────────────┘
```

Стиль:
- Красный фон с прозрачностью
- Красный текст
- Иконка ошибки
- Закругленные углы

## Индикаторы статуса

### Игрок
- **Активен**: Зеленая рамка
- **Пас**: Оранжевый бейдж
- **Перебор**: Красный бейдж
- **Победа**: Зеленый бейдж с 🏆
- **Проигрыш**: Красный бейдж с ❌

### Ход
- Подсветка текущего игрока
- Текст "Ваш ход" / "Ход игрока X"
- Пульсирующая анимация (планируется)

## Accessibility

- ✅ Контрастные цвета
- ✅ Крупные кнопки (min 44x44px)
- ✅ Четкие состояния
- ✅ Понятные иконки
- ⏳ Screen reader support (планируется)
- ⏳ Keyboard navigation (планируется)

## Темная тема (Telegram)

Все цвета адаптируются под тему Telegram:
- `var(--tg-theme-bg-color)` - фон
- `var(--tg-theme-text-color)` - текст
- `var(--tg-theme-button-color)` - кнопки
- `var(--tg-theme-hint-color)` - подсказки

## Производительность

- CSS Grid для раскладки
- Flexbox для выравнивания
- Минимум JavaScript для UI
- Оптимизированные изображения карт
- Lazy loading (планируется)
