# 📱 Компактный дизайн для мобильных устройств

## Проблема
Игрокам приходилось постоянно скроллить вверх-вниз, чтобы увидеть все элементы игры на мобильном экране.

## Решение
Оптимизирован дизайн для размещения всех элементов на одном экране без скролла.

## Основные изменения

### 1. Общая структура игры
**Было**:
- `min-height: 100vh` - могло быть больше экрана
- `padding: 1rem` - большие отступы
- `gap: 1rem` - большие промежутки

**Стало**:
- `height: 100vh` - строго по высоте экрана
- `padding: 0.5rem` - компактные отступы
- `gap: 0.5rem` - меньшие промежутки
- `overflow: hidden` - без скролла основного контейнера

### 2. Размеры карт
**Было**:
- Обычная карта: 80×112px
- Мини карта: 40×56px

**Стало**:
- Обычная карта: 60×84px (-25%)
- Мини карта: 32×45px (-20%)
- Меньшие шрифты и отступы

### 3. Секция других игроков
**Новое**:
- `max-height: 25vh` - максимум 25% экрана
- `overflow-y: auto` - скролл только для списка игроков
- Тонкий скроллбар (3px)
- Компактные карточки игроков

### 4. Уменьшенные отступы
**Все элементы**:
- Padding: 1rem → 0.75rem → 0.5rem
- Gap: 1rem → 0.5rem → 0.35rem
- Border-radius: 16px → 12px → 8px
- Font-size: уменьшены на 10-20%

### 5. Адаптивность по высоте
**Новое**: Media query для низких экранов
```css
@media (max-height: 700px) {
  /* Еще более компактные размеры */
  .other-players {
    max-height: 20vh;
  }
  /* Меньшие шрифты */
}
```

## Структура экрана

```
┌─────────────────────────────┐ ← 100vh (без скролла)
│ 💰 Баланс (компактный)      │ 5%
├─────────────────────────────┤
│ 🎩 Дилер + карты            │ 20%
├─────────────────────────────┤
│ 👥 Другие игроки            │ 25% (со скроллом)
│ (max-height + scroll)       │
├─────────────────────────────┤
│ 🎴 Ваши карты + инфо        │ 30%
├─────────────────────────────┤
│ 🎮 Кнопки управления        │ 15%
└─────────────────────────────┘
```

## Детальные изменения

### Game.css

#### Контейнер игры
```css
/* Было */
.game {
  min-height: 100vh;
  padding: 1rem;
  gap: 1rem;
}

/* Стало */
.game {
  height: 100vh;          /* Строго по экрану */
  padding: 0.5rem;        /* Меньше отступы */
  gap: 0.5rem;            /* Меньше промежутки */
  overflow: hidden;       /* Без скролла */
}
```

#### Баланс
```css
/* Было */
.game-balance {
  padding: 0.75rem 1rem;
  font-size: 1.1rem;
}

/* Стало */
.game-balance {
  padding: 0.5rem 0.75rem;  /* -33% */
  font-size: 0.95rem;       /* -14% */
  flex-shrink: 0;           /* Не сжимается */
}
```

#### Дилер
```css
/* Было */
.dealer-section {
  padding: 1rem;
  border-radius: 16px;
}

/* Стало */
.dealer-section {
  padding: 0.75rem;         /* -25% */
  border-radius: 12px;      /* -25% */
  flex-shrink: 0;           /* Не сжимается */
}
```

#### Другие игроки
```css
/* Новое */
.other-players {

  flex-shrink: 1;           /* Может сжиматься */
}

/* Тонкий скроллбар */

  width: 3px;
```

## Адаптивность

### Мобильные устройства (< 480px)
```css
@media (max-width: 480px) {
  .game {
    padding: 0.4rem;        /* Еще меньше */
    gap: 0.4rem;
  }
  
  .other-players {
    max-height: 22vh;       /* Меньше места */
  }
}
```

### Низкие экраны (< 700px высота)
```css
@media (max-height: 700px) {
  .game {
    gap: 0.35rem;           /* Минимальные промежутки */
  }
  
  .other-players {
    max-height: 20vh;       /* Еще меньше */
  }
  
  .score {
    font-size: 1.1rem;      /* Меньше шрифты */
  }
}
```
