# ✅ Улучшена видимость счета на цветном фоне

## Проблема
При смене цвета карточки игрока (зеленый/красный/серый) счет и текст становились плохо видны из-за низкого контраста.

## Решение
Добавлены CSS свойства для улучшения видимости:
1. `mix-blend-mode: difference` для счета
2. `text-shadow` для всех текстовых элементов
3. Принудительный белый цвет текста

## Изменения в CSS

### 1. Счет (score.large)
```css
/* Улучшенная видимость счета на цветном фоне */
.my-player.result-win .score.large,
.my-player.result-lose .score.large,
.my-player.result-push .score.large {
  mix-blend-mode: difference;
  color: white;
}
```

**Эффект**: Счет инвертируется относительно фона, всегда контрастный

### 2. Текстовые элементы
```css
/* Улучшенная видимость текста на цветном фоне */
.my-player.result-win .player-info,
.my-player.result-lose .player-info,
.my-player.result-push .player-info {
  text-shadow: 0 2px 4px rgba(0, 0, 0, 0.5);
}

.my-player.result-win .player-name,
.my-player.result-lose .player-name,
.my-player.result-push .player-name,
.my-player.result-win .my-bet,
.my-player.result-lose .my-bet,
.my-player.result-push .my-bet {
  color: white;
  text-shadow: 0 2px 4px rgba(0, 0, 0, 0.5);
}
```

**Эффект**: Текст белый с тенью, хорошо виден на любом фоне

## Как работает mix-blend-mode: difference

`mix-blend-mode: difference` инвертирует цвет элемента относительно фона:
- На зеленом фоне → светлый/контрастный
- На красном фоне → светлый/контрастный
- На сером фоне → светлый/контрастный

  Зеленый градиент
```

│ [🂡][🂮][🃇]        │

```

## Альтернативные решения (не использованы)

1. **Фиксированный цвет** - не адаптируется к фону
2. **Прозрачность** - ухудшает читаемость
3. **Обводка текста** - выглядит грубо
4. **Двойной текст** - избыточно
