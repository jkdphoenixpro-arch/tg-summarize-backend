# 💰 Система баланса и ставок

## Реализованная функциональность

### 1. Отображение баланса в лобби
- Карточка с текущим балансом игрока
- Загрузка баланса из MongoDB или дефолтное значение (1000₽)
- Красивый градиентный дизайн

### 2. Выбор ставки
- Минимальная ставка: 20₽
- Максимальная ставка: 300₽
- Кнопки +10/-10 для быстрого изменения
- Ручной ввод суммы ставки
- Проверка достаточности средств

### 3. Отображение в игре
- Баланс игрока в верхней части экрана
- Ставка каждого игрока под его именем
- Результат игры с указанием выигрыша/проигрыша:
  - 🎉 Вы выиграли +100₽!
  - 😔 Вы проиграли -100₽
  - 🤝 Ничья

### 4. Обновление баланса
- Автоматическое списание ставки при присоединении
- Обновление баланса после окончания игры:
  - Победа: баланс + ставка
  - Проигрыш: баланс - ставка
  - Ничья: баланс не меняется
- Синхронизация с MongoDB (если подключена)

## Компоненты

### Frontend
- `Lobby.jsx` - выбор ставки и отображение баланса
- `Lobby.css` - стили для лобби
- `Game.jsx` - отображение ставок и баланса в игре
- `Game.css` - стили для игрового экрана

### Backend
- `gameManager.js` - логика ставок и обновления баланса
- `models/Player.js` - MongoDB модель с балансом
- `server.js` - API endpoint `/api/player/:userId` для получения баланса

## API

### GET /api/player/:userId
Получить информацию об игроке

**Response:**
```json
{
  "success": true,
  "player": {
    "userId": "123456",
    "username": "Player",
    "balance": 1000,
    "gamesPlayed": 5,
    "gamesWon": 3
  }
}
```

## Проверка работы

1. Запустите игру через `/game21`
2. В лобби увидите свой баланс (1000₽ по умолчанию)
3. Выберите ставку (20-300₽)
4. Присоединитесь к игре
5. В игре увидите:
   - Свой баланс вверху экрана
   - Ставки всех игроков
6. После окончания игры:
   - Увидите результат с суммой выигрыша/проигрыша
   - Баланс автоматически обновится

## Без MongoDB

Если MongoDB не подключена:
- Используется дефолтный баланс 1000₽
- Баланс обновляется локально в памяти
- После перезапуска сервера баланс сбрасывается

## С MongoDB

При подключенной MongoDB:
- Баланс сохраняется в базе данных
- История игр записывается
- Статистика побед/поражений
- Баланс сохраняется между сессиями
