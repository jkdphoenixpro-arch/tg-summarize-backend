# 🔄 Диаграмма работы системы баланса и ставок

## Общий поток

```
┌─────────────┐
│   Игрок     │
│  открывает  │
│   /game21   │
└──────┬──────┘
       │
       ▼
┌─────────────────────────────┐
│  Telegram Bot               │
│  отправляет WebApp кнопку   │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  React App (Lobby)          │
│  загружается                │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  GET /api/player/:userId    │
│  получает баланс            │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Отображение лобби:         │
│  - Баланс: 1000₽           │
│  - Выбор ставки: 20-300₽   │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Игрок выбирает ставку      │
│  и нажимает "Присоединиться"│
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Socket: join_game          │
│  { gameId, userId, bet }    │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Server проверяет:          │
│  - Баланс >= ставка?        │
│  - Ставка в лимитах?        │
└──────┬──────────────────────┘
       │
       ├─── ❌ Ошибка
       │    └─> Сообщение об ошибке
       │
       └─── ✅ OK
            │
            ▼
┌─────────────────────────────┐
│  Игрок добавлен в игру      │
│  с балансом и ставкой       │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Отображение игры:          │
│  - Баланс вверху            │
│  - Ставки игроков           │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Игра идет...               │
│  (hit, stand, etc.)         │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Игра завершена             │
│  determineWinners()         │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Для каждого игрока:        │
│  - Определить результат     │
│  - Обновить баланс в MongoDB│
│  - Обновить баланс в state  │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Отображение результата:    │
│  🎉 Вы выиграли +100₽!     │
│  💰 Баланс: 1100₽          │
└─────────────────────────────┘
```

## Проверка баланса при присоединении

```
┌─────────────────────────────┐
│  join_game(bet: 100)        │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Проверка ставки            │
│  bet >= MIN_BET (20)?       │
│  bet <= MAX_BET (300)?      │
└──────┬──────────────────────┘
       │
       ├─── ❌ Нет
       │    └─> "Неверная ставка"
       │
       └─── ✅ Да
            │
            ▼
┌─────────────────────────────┐
│  MongoDB подключена?        │
└──────┬──────────────────────┘
       │
       ├─── ❌ Нет
       │    └─> Использовать дефолт 1000₽
       │
       └─── ✅ Да
            │
            ▼
┌─────────────────────────────┐
│  Player.getOrCreate()       │
│  Получить баланс из БД      │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  balance >= bet?            │
└──────┬──────────────────────┘
       │
       ├─── ❌ Нет
       │    └─> "Недостаточно средств"
       │
       └─── ✅ Да
            │
            ▼
┌─────────────────────────────┐
│  Добавить игрока в игру     │
│  с балансом и ставкой       │
└─────────────────────────────┘
```

## Обновление баланса после игры

```
┌─────────────────────────────┐
│  Игра завершена             │
│  determineWinners()         │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Для каждого игрока:        │
│  Определить результат       │
│  (win/lose/push)            │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  MongoDB подключена?        │
└──────┬──────────────────────┘
       │
       ├─── ❌ Нет
       │    │
       │    ▼
       │    ┌─────────────────────┐
       │    │ Обновить локально:  │
       │    │ win: balance + bet  │
       │    │ lose: balance - bet │
       │    │ push: balance       │
       │    └─────────────────────┘
       │
       └─── ✅ Да
            │
            ▼
┌─────────────────────────────┐
│  Player.updateAfterGame()   │
│  Обновить баланс в БД       │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Получить новый баланс      │
│  из БД                      │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Обновить баланс в          │
│  gameState для отображения  │
└──────┬──────────────────────┘
       │
       ▼
┌─────────────────────────────┐
│  Socket: game_over          │
│  Отправить обновленное      │
│  состояние клиенту          │
└─────────────────────────────┘
```

## Состояния игрока

```
┌─────────────────────────────────────────┐
│  Жизненный цикл игрока                  │
├─────────────────────────────────────────┤
│                                         │
│  1. Создание                            │
│     {                                   │
│       userId, username, photoUrl,       │
│       cards: [],                        │
│       score: 0,                         │
│       status: 'active',                 │
│       bet: 100,                         │
│       balance: 1000                     │
│     }                                   │
│                                         │
│  2. Игра                                │
│     status: 'active' | 'stand' | 'busted'│
│     cards: [...карты...]                │
│     score: 0-21+                        │
│                                         │
│  3. Завершение                          │
│     result: 'win' | 'lose' | 'push'     │
│     balance: обновлен                   │
│                                         │
└─────────────────────────────────────────┘
```

## Обработка ошибок

```
┌─────────────────────────────────────────┐
│  Возможные ошибки                       │
├─────────────────────────────────────────┤
│                                         │
│  ❌ "Игра не найдена"                   │
│     → Неверный gameId                   │
│                                         │
│  ❌ "Игра уже началась"                 │
│     → Нельзя присоединиться             │
│                                         │
│  ❌ "Игра заполнена"                    │
│     → Максимум 6 игроков                │
│                                         │
│  ❌ "Минимальная ставка 20₽"            │
│     → bet < MIN_BET                     │
│                                         │
│  ❌ "Максимальная ставка 300₽"          │
│     → bet > MAX_BET                     │
│                                         │
│  ❌ "Недостаточно средств"              │
│     → balance < bet                     │
│                                         │
│  ❌ "Не ваш ход"                        │
│     → Попытка действия не в свой ход    │
│                                         │
└─────────────────────────────────────────┘
```
