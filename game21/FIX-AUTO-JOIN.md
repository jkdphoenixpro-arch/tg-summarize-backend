# ✅ Исправлено: Автоматическое присоединение к игре

## Проблема
При открытии игры через `/game21` игрок сразу присоединялся к игре со ставкой 20₽, не видя лобби с выбором ставки.

## Решение
Убрано автоматическое присоединение. Теперь игрок всегда видит лобби и может выбрать ставку перед присоединением.

## Что изменилось

### 1. App.jsx
- ❌ Удален useEffect с автоматическим присоединением
- ✅ Теперь всегда показывается лобби, если нет gameState
- ✅ gameId предзаполняется в поле, но не происходит автоматический join

### 2. Lobby.jsx
- ✅ Добавлен useEffect для обновления gameId из props
- ✅ Добавлена подсказка когда gameId предзаполнен
- ✅ Игрок должен вручную выбрать ставку и нажать "Присоединиться"

### 3. Lobby.css
- ✅ Добавлен стиль для hint-text

## Как это работает теперь

### Сценарий 1: Открытие через /game21
```
1. Пользователь нажимает /game21
2. Бот отправляет WebApp с gameId в URL
3. Открывается лобби с предзаполненным gameId
4. Показывается подсказка: "ID игры предзаполнен. Выберите ставку и присоединяйтесь!"
5. Игрок видит свой баланс
6. Игрок выбирает ставку (20-300₽)
7. Игрок нажимает "Присоединиться"
8. Только теперь происходит join_game
```

### Сценарий 2: Прямое открытие WebApp
```
1. Пользователь открывает WebApp напрямую
2. Открывается лобби с пустым полем gameId
3. Игрок вводит gameId вручную
4. Игрок выбирает ставку
5. Игрок нажимает "Присоединиться"
```

## Преимущества

✅ Игрок всегда видит свой баланс перед игрой
✅ Игрок может выбрать любую ставку (20-300₽)
✅ Проверка достаточности средств перед присоединением
✅ Более понятный UX - явное действие "Присоединиться"
✅ Можно изменить ставку перед присоединением

## Код изменений

### App.jsx - Удалено
```javascript
// ❌ УДАЛЕНО - автоматическое присоединение
useEffect(() => {
    if (socket && user && gameId && !gameState) {
        socket.emit('join_game', {
            gameId: gameId,
            userId: user.userId,
            username: user.username,
            photoUrl: user.photoUrl,
            bet: 20
        });
    }
}, [socket, user, gameId, gameState]);
```

### App.jsx - Изменено
```javascript
// ✅ ИЗМЕНЕНО - всегда показывать лобби если нет gameState
if (!gameState) {
    return (
        <Lobby 
            onJoinGame={handleJoinGame} 
            initialGameId={gameId}
            user={user}
            socketUrl={SOCKET_URL}
        />
    );
}
```

### Lobby.jsx - Добавлено
```javascript
// ✅ ДОБАВЛЕНО - обновление gameId из props
useEffect(() => {
    if (initialGameId && initialGameId !== gameId) {
        setGameId(initialGameId);
    }
}, [initialGameId]);

// ✅ ДОБАВЛЕНО - подсказка в JSX
{initialGameId && (
    <p className="hint-text">
        ID игры предзаполнен. Выберите ставку и присоединяйтесь!
    </p>
)}
```
