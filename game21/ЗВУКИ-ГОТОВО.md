## Особенности реализации

### Умное воспроизведение

1. **Начало игры** - отслеживается изменение статуса через `useRef`
2. **Взятие карты** - отслеживается увеличение количества карт (только после начальной раздачи)
3. **Результат** - отслеживается появление result через `useRef`
4. **Таймаут** - через событие socket.io

### Предотвращение дублирования

Используются `useRef` для хранения предыдущих значений:
- `prevGameStatusRef` - предыдущий статус игры
- `prevMyCardsLengthRef` - предыдущее количество карт
- `prevResultRef` - предыдущий результат

Это предотвращает повторное воспроизведение звуков при ре-рендерах.

## Управление звуками

### В коде:
```javascript
import soundManager from '../utils/sounds';

// Включить/выключить
soundManager.toggle();

// Установить громкость
soundManager.setVolume(0.7); // 70%

// Проверить статус
console.log(soundManager.enabled); // true/false
```

### Настройки сохраняются

Настройки автоматически сохраняются в `localStorage`:
- Громкость
- Включен/выключен

## Настройка громкости

Если звуки слишком громкие/тихие, отредактируй `src/utils/sounds.js`:

```javascript
// Общая громкость
this.volume = 0.5; // 50% (по умолчанию)

// Или для отдельных звуков
this.sounds.win.volume = 0.8;  // Победа громче
this.sounds.lose.volume = 0.4; // Проигрыш тише
```

## Возможные улучшения (опционально)

### 1. Кнопка управления звуком в UI

```javascript
function SoundToggle() {
  const [enabled, setEnabled] = useState(soundManager.enabled);
  
  const toggle = () => {
    const newState = soundManager.toggle();
    setEnabled(newState);
  };
  
  return (
    <button onClick={toggle} className="sound-btn">
      {enabled ? '🔊' : '🔇'}
    </button>
  );
}
```

### 2. Слайдер громкости

```javascript
function VolumeSlider() {
  const [volume, setVolume] = useState(soundManager.volume);
  
  const handleChange = (e) => {
    const newVolume = parseFloat(e.target.value);
    soundManager.setVolume(newVolume);
    setVolume(newVolume);
  };
  
  return (
    <input 
      type="range" 
      min="0" 
      max="1" 
      step="0.1" 
      value={volume}
      onChange={handleChange}
    />
  );
}
```
