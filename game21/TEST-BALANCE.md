# 🧪 Тестирование системы баланса и ставок

## Быстрый тест

### 1. Запустите игру
```bash
# Из корня проекта
npm run dev
```

### 2. Откройте игру в Telegram
```
/game21
```

### 3. Проверьте лобби
✅ Должны увидеть:
- Карточку с балансом (1000₽ по умолчанию)
- Поле для ввода ID игры
- Выбор ставки (20-300₽)
- Кнопки +10/-10 для изменения ставки
- Кнопку "Присоединиться" с суммой ставки

### 4. Выберите ставку
- Попробуйте изменить ставку кнопками +10/-10
- Попробуйте ввести ставку вручную
- Проверьте, что нельзя поставить меньше 20₽
- Проверьте, что нельзя поставить больше 300₽
- Проверьте, что нельзя поставить больше баланса

### 5. Присоединитесь к игре
- Введите любой ID игры (например: test123)
- Нажмите "Присоединиться"

### 6. Проверьте игровой экран
✅ Должны увидеть:
- Баланс вверху экрана (💰 Баланс: 1000₽)
- Вашу ставку под именем (Ставка: 100₽)
- Ставки других игроков (если есть)

### 7. Сыграйте игру
- Дождитесь второго игрока или откройте в другой вкладке
- Начните игру
- Сыграйте раунд

### 8. Проверьте результат
✅ После окончания игры должны увидеть:
- Результат с суммой: "🎉 Вы выиграли +100₽!" или "😔 Вы проиграли -100₽"
- Обновленный баланс вверху экрана

## Тест с MongoDB

### 1. Убедитесь что MongoDB запущена
```bash
# Проверка подключения
mongosh
```

### 2. Проверьте переменные окружения
В `server/.env` должно быть:
```env
MONGODB_URI=mongodb://localhost:27017/game21
```

### 3. Сыграйте несколько игр
- Выиграйте игру - баланс должен увеличиться
- Проиграйте игру - баланс должен уменьшиться

### 4. Проверьте базу данных
```bash
mongosh
use game21
db.players.find().pretty()
```

✅ Должны увидеть:
- Ваш userId
- Текущий баланс
- Количество игр (gamesPlayed)
- Количество побед (gamesWon)

### 5. Перезапустите сервер
```bash
# Ctrl+C и снова npm run dev
```

### 6. Откройте игру снова
✅ Баланс должен сохраниться!

## Возможные проблемы

### Баланс не отображается
- Проверьте консоль браузера (F12)
- Проверьте что сервер запущен
- Проверьте SOCKET_URL в client/.env

### Баланс не сохраняется
- Проверьте подключение к MongoDB
- Проверьте логи сервера
- Убедитесь что MONGODB_URI правильный

### Ошибка "Недостаточно средств"
- Это нормально если баланс меньше ставки
- Уменьшите ставку или пополните баланс в БД

### Ставка не отображается в игре
- Перезагрузите страницу
- Проверьте что gameManager.js обновлен
- Проверьте консоль на ошибки
