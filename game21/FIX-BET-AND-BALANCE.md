# ✅ Исправлено: Ставка и баланс

## Проблемы
1. ❌ Ставка не передавалась на сервер (всегда показывало 20₽)
2. ❌ Баланс не вычитался при присоединении к игре

## Решение

### 1. Передача ставки на сервер
**Файл**: `server/server.js`

**Было**:
```javascript
socket.on('join_game', async ({ gameId, userId, username, photoUrl }) => {
  const result = await gameManager.joinGame(gameId, userId, username, photoUrl);
```

**Стало**:
```javascript
socket.on('join_game', async ({ gameId, userId, username, photoUrl, bet }) => {
  console.log('🎮 join_game event:', { gameId, userId, username, bet });
  const result = await gameManager.joinGame(gameId, userId, username, photoUrl, bet);
```

### 2. Списание ставки при присоединении
**Файл**: `server/gameManager.js`

**Добавлено в метод `joinGame`**:
```javascript
// Списываем ставку сразу при присоединении
if (isDatabaseConnected()) {
  player.balance -= bet;
  await player.save();
  playerBalance = player.balance;
  console.log(`💰 Списано ${bet}₽ с баланса ${username}. Новый баланс: ${playerBalance}₽`);
} else {
  // Если MongoDB не подключена, вычитаем локально
  playerBalance -= bet;
}
```

### 3. Изменена логика выплат
**Файл**: `server/gameManager.js`

**Было**: Вычитали/добавляли ставку в конце игры

**Стало**: Ставка уже списана, теперь:
- **Победа**: баланс + (ставка × 2) - возврат ставки + выигрыш
- **Проигрыш**: ничего не делаем - ставка уже списана
- **Ничья**: баланс + ставка - возврат ставки

```javascript
if (player.result === 'win') {
  // Возвращаем ставку + выигрыш
  dbPlayer.balance += player.bet * 2;
  dbPlayer.gamesWon += 1;
} else if (player.result === 'push') {
  // Возвращаем ставку
  dbPlayer.balance += player.bet;
}
// При lose ничего не делаем - ставка уже списана
```

## Как это работает теперь

### Сценарий: Игрок ставит 70₽

#### 1. Присоединение к игре
```
Начальный баланс: 1000₽
Ставка: 70₽
→ Списывается 70₽
→ Баланс в игре: 930₽
```

#### 2. Игра идет
```
Баланс остается: 930₽
Ставка отображается: 70₽
```

#### 3. Результат игры

**Победа**:
```
Баланс: 930₽
→ Добавляется 70₽ × 2 = 140₽
→ Новый баланс: 1070₽
→ Чистый выигрыш: +70₽
```

**Проигрыш**:
```
Баланс: 930₽
→ Ничего не добавляется
→ Новый баланс: 930₽
→ Чистый проигрыш: -70₽
```

**Ничья**:
```
Баланс: 930₽
→ Возвращается 70₽
→ Новый баланс: 1000₽
→ Чистый результат: 0₽
```
