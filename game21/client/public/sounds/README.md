# Звуковые эффекты для игры Блекджек 🔊

## Добавленные звуковые файлы

✅ Все звуки добавлены и интегрированы в игру!

### Список звуков:

| Файл | Описание | Когда воспроизводится |
|------|----------|----------------------|
| ✅ `card-flick.wav` | Звук взятия карты | Когда игрок берет карту (Hit) |
| ✅ `game-ready.wav` | Звук начала игры | Когда игра начинается |
| ✅ `pass.wav` | Звук паса | Когда игрок делает пас (Stand) |
| ✅ `win.wav` | Звук победы | Когда игрок выигрывает |
| ✅ `lose.wav` | Звук проигрыша | Когда игрок проигрывает |

## Интеграция

Звуки автоматически воспроизводятся в следующих ситуациях:

1. **🎮 Начало игры** → `game-ready.wav`
2. **🃏 Взятие карты** → `card-flick.wav`
3. **✋ Пас** → `pass.wav`
4. **⏰ Таймаут** → `pass.wav` (автоматический пас)
5. **🎉 Победа** → `win.wav`
6. **😔 Проигрыш** → `lose.wav`

## Управление звуками

Звуки можно:
- Включить/выключить через `soundManager.toggle()`
- Настроить громкость через `soundManager.setVolume(0.5)`
- Настройки сохраняются в localStorage

## Тестирование

Запусти игру и проверь:
- ✅ Звук при начале игры
- ✅ Звук при взятии карты
- ✅ Звук при пасе
- ✅ Звук при победе/проигрыше
- ✅ Звук при таймауте

Всё готово! 🎵
