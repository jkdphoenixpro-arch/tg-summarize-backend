# 📝 Changelog - Система баланса и ставок

## Дата: 09.02.2026

### ✨ Новые возможности

#### 1. Система баланса игроков
- Каждый игрок имеет баланс (по умолчанию 1000₽)
- Баланс отображается в лобби и во время игры
- Баланс сохраняется в MongoDB (опционально)
- Без MongoDB баланс хранится в памяти

#### 2. Выбор ставки в лобби
- Минимальная ставка: 20₽
- Максимальная ставка: 300₽
- Кнопки быстрого изменения (+10/-10)
- Ручной ввод суммы
- Проверка достаточности средств
- Блокировка кнопки при недостатке средств

#### 3. Отображение ставок в игре
- Ставка каждого игрока под его именем
- Ставка текущего игрока в карточке
- Результат игры с указанием суммы выигрыша/проигрыша

#### 4. Автоматическое обновление баланса
- При победе: баланс + ставка
- При проигрыше: баланс - ставка
- При ничьей: баланс не меняется
- Синхронизация с MongoDB
