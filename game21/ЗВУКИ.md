## Как работает

### Автоматические звуки:

- **🎮 Начало игры** → `game-ready.wav` играет когда статус меняется на 'playing'
- **🃏 Взятие карты** → `card-flick.wav` играет при нажатии "Взять карту"
- **✋ Пас** → `pass.wav` играет при нажатии "Пас"
- **⏰ Таймаут** → `pass.wav` играет при автоматическом пасе
- **🎉 Победа** → `win.wav` играет когда result === 'win'
- **😔 Проигрыш** → `lose.wav` играет когда result === 'lose'

### Управление звуками:

```javascript
import soundManager from '../utils/sounds';

// Включить/выключить
soundManager.toggle();

// Установить громкость (0.0 - 1.0)
soundManager.setVolume(0.7);

// Проверить статус
console.log(soundManager.enabled); // true/false
```

## Настройка громкости

Если нужно изменить громкость, отредактируй `src/utils/sounds.js`:

```javascript
this.volume = 0.5; // По умолчанию 50%
```

Или для отдельных звуков:
```javascript
this.sounds.win.volume = 0.8;  // Победа громче
this.sounds.lose.volume = 0.4; // Проигрыш тише
```
