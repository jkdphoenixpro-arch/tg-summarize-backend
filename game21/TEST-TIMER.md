# Тестирование таймера хода

## Быстрый запуск для тестирования

### 1. Запустите сервер
```bash
cd game21/server
npm start
```

### 2. Запустите клиент
```bash
cd game21/client
npm run dev
```

### 3. Тестовый сценарий

#### Сценарий 1: Нормальный ход
1. Откройте игру в 2+ вкладках браузера (разные игроки)
2. Присоединитесь к игре со всех вкладок
3. Начните игру
4. Наблюдайте таймер - он должен показывать 30 секунд
5. Сделайте ход (Hit или Stand) до истечения времени
6. Таймер должен сброситься для следующего игрока

#### Сценарий 2: Таймаут
1. Откройте игру в 2+ вкладках
2. Присоединитесь и начните игру
3. **НЕ ДЕЛАЙТЕ НИКАКИХ ДЕЙСТВИЙ** 30 секунд
4. Должно произойти:
   - Таймер достигнет 0
   - Появится сообщение "⏰ Время вышло! Автоматический пас"
   - Автоматически произойдет "Stand"
   - Ход перейдет к следующему игроку

#### Сценарий 3: Визуальные эффекты
1. Наблюдайте за цветом таймера:
   - **Зеленый** (> 10 секунд) - спокойное состояние
   - **Оранжевый** (5-10 секунд) - предупреждение, легкая пульсация
   - **Красный** (< 5 секунд) - критическое состояние, интенсивная пульсация

## Что проверить

- ✅ Таймер отображается во время игры
- ✅ Таймер обновляется каждую секунду
- ✅ Цвет меняется в зависимости от времени
- ✅ Анимация пульсации работает
- ✅ При таймауте автоматически происходит Stand
- ✅ Показывается сообщение о таймауте
- ✅ Ход переходит к следующему игроку
- ✅ Таймер сбрасывается для нового игрока
- ✅ При действии Hit таймер перезапускается
- ✅ При действии Stand таймер останавливается

## Настройка времени для быстрого тестирования

Если хотите протестировать быстрее, можно временно уменьшить время:

В файле `game21/server/gameManager.js`:
```javascript
this.TURN_TIMEOUT = 10000; // 10 секунд вместо 30
```

**Не забудьте вернуть обратно после тестирования!**

## Возможные проблемы

### Таймер не отображается
- Проверьте что сервер запущен
- Проверьте консоль браузера на ошибки
- Убедитесь что игра в статусе 'playing'

### Таймер не обновляется
- Проверьте WebSocket соединение
- Посмотрите логи сервера
- Убедитесь что интервал обновления запущен

### Таймаут не срабатывает
- Проверьте логи сервера - должно быть сообщение "⏰ Время вышло для игры..."
- Убедитесь что таймер не был очищен преждевременно
- Проверьте что игра в статусе 'playing'
