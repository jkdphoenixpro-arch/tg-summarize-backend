# Настройка Userbot для сбора истории сообщений 🔧

Userbot использует твой личный аккаунт Telegram для доступа к истории сообщений.

## ⚠️ Важно!

- Userbot работает от твоего имени (не как бот)
- Может читать всю историю сообщений
- Требует API ключи от Telegram
- Безопасно, если используешь правильно

## Шаг 1: Получи API ключи

1. Зайди на https://my.telegram.org/auth
2. Войди с номером телефона
3. Перейди в "API development tools"
4. Создай приложение:
   - App title: `Message Collector`
   - Short name: `collector`
   - Platform: `Desktop`
5. Скопируй:
   - `api_id` (число)
   - `api_hash` (строка)

## Шаг 2: Установи зависимости

```bash
npm install telegram input
```

## Шаг 3: Настрой .env

Добавь в `.env`:

```env
# Userbot настройки (получи на https://my.telegram.org)
TELEGRAM_API_ID=твой_api_id
TELEGRAM_API_HASH=твой_api_hash
TELEGRAM_SESSION=
TARGET_CHAT_ID=-1002438653104
MESSAGE_LIMIT=100
```

## Шаг 4: Первый запуск (авторизация)

```bash
node userbot-collector.js
```

Тебя попросят:
1. Ввести номер телефона (с +)
2. Ввести код из Telegram
3. Ввести пароль 2FA (если есть)

После успешной авторизации скопируй строку сессии и добавь в `.env`:
```
TELEGRAM_SESSION=длинная_строка_сессии
```

## Шаг 5: Использование

После настройки просто запускай:

```bash
node userbot-collector.js
```

Userbot:
- Подключится к твоему аккаунту
- Соберет последние N сообщений из указанного чата
- Сохранит в файл `history_CHATID_TIMESTAMP.txt`

## Настройки в .env

```env
# ID чата (можно узнать у бота @userinfobot)
TARGET_CHAT_ID=-1002438653104

# Сколько сообщений собрать (максимум)
MESSAGE_LIMIT=1000
```

## Как узнать ID чата?

1. **Через бота @userinfobot:**
   - Перешли сообщение из нужного чата боту
   - Он покажет ID чата

2. **Через веб-версию:**
   - Открой чат в web.telegram.org
   - ID в URL: `https://web.telegram.org/k/#-1002438653104`

3. **Через твоего бота:**
   - Добавь бота в чат
   - Посмотри в логах: `Чат ID: -1002438653104`

## Безопасность

✅ **Безопасно:**
- Сессия хранится локально в `.env`
- Используешь официальный Telegram API
- Не передаешь данные третьим лицам

⚠️ **Не делай:**
- Не публикуй `.env` файл
- Не передавай `TELEGRAM_SESSION` никому
- Не запускай на чужих серверах

## Альтернатива: Telegram Desktop

Если не хочешь настраивать Userbot:

1. Открой Telegram Desktop
2. Зайди в нужный чат
3. Меню (⋮) → Export chat history
4. Выбери формат: JSON или TXT
5. Укажи количество сообщений
6. Экспорт готов!

## Troubleshooting

**Ошибка "Phone number invalid":**
- Вводи номер с кодом страны: `+79991234567`

**Ошибка "Session expired":**
- Удали `TELEGRAM_SESSION` из `.env`
- Запусти снова для новой авторизации

**Ошибка "Chat not found":**
- Проверь правильность ID чата
- Убедись что ты участник этого чата

**Ошибка "Too many requests":**
- Telegram ограничивает частоту запросов
- Подожди 1-2 минуты и попробуй снова
